SAP to Azure Secure Data Pipeline

Developed a Node.js/Express microservice on GCP to establish a secure file upload pipeline between SAP ABAP and Azure Blob Storage. Engineered the flow to process multipart/form-data via Multer, implemented secure credential retrieval from Hashicorp Vault for Azure SDK authentication, and managed cross-cloud data streaming.
