Personal Project

Erratic Drummer

View on Play Store GitHub Source

A native Android application engineered to generate dynamic, erratic drum beats. Developed entirely from scratch, this project served as a deep dive into Android mobile development, handling real-time audio generation, UI/UX design, and the complete Google Play Store publication lifecycle.

Live Demo

Core Functionality

  • ▹ Audio Engine: Engineered reliable playback systems for generating erratic, randomized drum patterns.
  • ▹ User Interface: Designed an intuitive, touch-friendly interface allowing users to control parameters and interact with the beat generator seamlessly.
  • ▹ State Management: Handled complex application states to ensure uninterrupted audio playback during backgrounding or screen rotations.

Technical Stack & Deployment

  • ▹ Android SDK: Built using native Android development tools and libraries for optimal performance.
  • ▹ Version Control: Maintained clean, documented source code on GitHub.
  • ▹ Play Console: Navigated the Google Play Console for app signing, asset generation, policy compliance, and production rollout.
