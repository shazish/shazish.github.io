Pythia A Kalshi-Polymarket Opportunity Finder

Autonomous pipeline that scans 10,000+ Kalshi and Polymarket prediction markets, runs multi-agent web research on candidates, classifies near-certain outcomes with an LLM, fact-checks every CERTAIN result, and outputs a Kelly-sized opportunity report.

shazish/pythia-kalshi-pm-finder
Fact-checked outputs
Every CERTAIN classification is automatically re-verified against sources. Hallucinations get downgraded to LIKELY before the report is written.
Kelly-sized positions
Position sizes are computed per conviction level using the Kelly criterion, capped at 5% of bankroll. Edge is shown net of Kalshi and Polymarket fees.
Checkpoint / resume
Classification state is saved after every ticker. Interrupt and restart at any point — already-classified markets are skipped on the next run.
Free research model
Web research runs on Owl Alpha via OpenRouter's free tier. No API cost per candidate. Only the final classification step uses a paid LLM call.

Technical Details

Architecture

  • Entry point: kalshi-pm-analyzer (single orchestrator)
  • Research: Owl Alpha subagents via OpenRouter (free tier)
  • Classification: scripts/classify_all.py — checkpoint/resume, saves after every ticker
  • Verification: scripts/verify_classifications.py — hallucination + source checks
  • Output: Excel report + pipeline_run.md log per run
  • Anomaly mode: volume-first scanner detects smart-money signals below 80¢

Edge & Sizing

  • Kalshi fee: profit-based ~1.5% on winnings
  • Polymarket fee: volume-based, per-category taker rates
  • Threshold: min 3% edge after fees, 15% annualized
  • Kelly cap: max 5% of bankroll per position
  • Classification: CERTAIN / LIKELY / UNCLEAR with confidence score
  • Runtime: ~60–90 min for 150 candidates (owl-alpha)
