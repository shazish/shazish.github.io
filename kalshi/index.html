Pythia

Prediction Intelligence for
Kalshi & Polymarket

Autonomous pipeline that scans 10,000+ prediction markets, runs multi-agent web research on candidates, classifies near-certain outcomes with an LLM, fact-checks every CERTAIN result, and outputs a Kelly-sized opportunity report.

Why Pythia
Built-in safeguards, not afterthoughts
Fact-checked
Every CERTAIN classification is automatically re-verified against sources. Hallucinations are downgraded to LIKELY before the report is written.
Kelly sizing
Position sizes computed per conviction level using the Kelly criterion, capped at 5% of bankroll. Edge shown net of Kalshi and Polymarket fees.
Checkpointing
Classification state is saved after every ticker. Interrupt and restart at any point — already-classified markets are skipped automatically.
Low cost
Web research runs on Owl Alpha via OpenRouter's free tier. No API cost per candidate. Only the final classification step uses a paid LLM call.
How it works
Pipeline
pythia-main {mode}
Scan
Price + volume filters. No LLM. ~150 candidates from 10K markets.
Research
Owl Alpha subagents run web searches in sequential batches of ~50.
Classify
LLM classifies each ticker using pre-conducted evidence. Checkpoint/resume.
Verify
Auto fact-checks CERTAIN entries. Downgrades hallucinations to LIKELY.
Report
Excel report with Kelly-sized positions and edge after fees.
